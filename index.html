Twitter Ego Network

An analysis of a twitter follower network.

The Network

Unweighted directed graph.

Edge “follower” (a-->b – a is followed by b).

Initially:

  • 404.494 edges
  • 367.671 nodes

Expectations

  • New users tend to follow big mainstream figures
  • Scale-free network?

First Analysis

Stats

  • Number of Nodes: 367.671
  • Number of Edges: 404.494
  • Clustering Coefficient: 0.008
  • Average Degree: 1.1
  • Network Diameter: 7
  • Average Path Length: 3.936
  • Max Degree: 82/49846 (IN/OUT)
  • Min Degree: 1/1 (IN/OUT)
  • 210 users with a fully “crawled” profile from the full set of 367.671.

Conclusions
